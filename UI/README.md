# Knowledge Store Explorer

A simple, interactive web UI for browsing Open Knowledge Format (OKF) bundles generated by the Knowledge Builder pipeline.

## Features

- Select any document bundle under `knowledge_store/`.
- **Overview**: page counts, concept counts, relationship counts, metadata.
- **Concepts**: searchable list of extracted concepts with detail modal.
- **Graph**: interactive 3D/2D force-directed graph of concept relationships (WebGL via `3d-force-graph`, canvas via `force-graph`), with community clustering, neighborhood highlighting, search, and cluster filters. Requires `npm install` once inside `UI/` to fetch the graph libraries (served from `node_modules/`).
- **Pages**: per-page extracted content.
- **Indexes**: keyword, alias, glossary, and page indexes.

## Run

From the project root:

```bash
python UI/server.py
```

Then open [http://localhost:8000](http://localhost:8000) in your browser.

The server uses only Python's standard library (`http.server`) and has no external dependencies.

## Customize

Change the port with the `PORT` environment variable:

```bash
PORT=9000 python UI/server.py
```
